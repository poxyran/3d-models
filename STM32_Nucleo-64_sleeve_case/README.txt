# Summary

Sleeve case for the [STM32 Nucleo-64](http://www.st.com/en/evaluation-tools/stm32-mcu-nucleo.html?querycriteria=productId=LN1847) development boards.

Main features:
* accessible Arduino and top/bottom Morpho headers
* visible onboard LEDs
* USER and RESET pushbuttons spring cutout

This thing is inspired by the [BeagleBone Black sleeve case](https://www.thingiverse.com/thing:2365555) and the [Sleeve Case for Raspberry Pi B+](https://www.thingiverse.com/thing:604915)
